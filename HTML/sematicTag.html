두꺼워져라

<b>태그는 css처럼 단순하게 웹브라우저에 표시되는 모양만 결정한다
CSS에서 폰트 두께는 Font-weight라는 속성으로 조절한다

두꺼워져라

<strong> 태그는 simantic tag로 html 문서 내에서 강조하고 싶은 내용에 사용한다

둘 다 폰트가 두꺼워 지지만 strong은 내용을 강조

<i>태그는 CSS처럼 웹 사이트에 표시되는 모양만 결정 이탤릭체
내용강조 없이 이탤릭체로 표현 : CSS font-style 속성을 사용

<em>태그는 Semantic Tag로 html에서 이탤릭체로 내용을 강조할때 싶을 때 사용한다

<s>태그는 CSS처럼 웹 사이트에 표시되는 모양만 결정 취소선
취소된 내용이라는 강조가 아닌 중간의 가로선 : text-decoration 속성을 사용

<del>태그는 Semantic Tag로 html에서 삭제된 내용을 강조하고 싶을 때 사용한다

<u>태그는 웹 브라우저에 표시되는 모양만 결정 언더라인 CSS text decoration : underline

<ins> 태그는 semantic tag로 html에서 내용에 밑줄을 만들어서 새로 삽입된 내용을 강조하고 싶을 때 사용한다.
